# Projeto Slide

## Tela de LOGIN

### Login: Administrador 
* Criar cadastro de imagem com Titulo e descrição.
* Fazer consulta
* Editar cadastro
* Deletar cadastro.

### Login: Usuário

* Apenas visualizar o cadastro (Imagem, Titulo e Descrição).


### As tecnologias utilizadas neste projeto foram:
* HTML5
* CSS3
* JAVASCRIPT
* PHP
* Banco de dados MYSQL
